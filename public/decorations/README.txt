Drop your chosen decorative art into this folder using these exact filenames
(the code already references these paths — no code changes needed once
they're here):

  siren-harp.png          -> shown above the Menu section (tavern food/drink)
                              on building pages and the building edit form
  warriors-standards.webp -> shown at the top of the Residents tab
  knight-horseback.webp   -> shown on the DM Login screen
  sea-serpent.jpeg        -> shown at the top of the Buildings list tab

Any image format works (png/webp/jpeg/svg) as long as the filename and
extension match exactly what's listed above. If a file is missing, that
decoration slot just quietly doesn't render — nothing breaks.

For best results, use images with a transparent or white background —
they're displayed at a modest size with a subtle sepia tint to blend with
the parchment/ink theme.
